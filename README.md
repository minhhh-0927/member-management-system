# Member Management System

## Installation

```bash
$ yarn install
```

## Migration file
Migrate

```shell script
yarn build
ts-node ./node_modules/typeorm/cli.js migration:run
```

## Running the app

```bash
# create file .env 
$ cp .env.example .env
# development
$ yarn start

# watch mode
$ yarn start:dev

# incremental rebuild (webpack)
$ yarn webpack
$ yarn start:hmr

# production mode
$ yarn start:prod
```

## Test

```bash
# unit tests
$ yarn test

# e2e tests
$ yarn test:e2e

# test coverage
$ yarn test:cov
```
